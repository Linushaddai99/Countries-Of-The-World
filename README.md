# 🌏Countries Of The World <a name="about-project"></a>

In this web application, users are able to see all the countries in the world on the homepage, filter countries by region, click on a country to see more information, search for a country and toggle dark and light modes.

## 💻 Getting Started <a name="getting-started"></a>

<!-- > Describe how a new developer could make use of your project. -->

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
Install nodejs
```

### Setup

Clone this repository to your desired folder:

```sh
git clone https://github.com/Linushaddai99/Countries-Of-The-World.git
```

### Install

Install this project with:

```sh
cd Countries-Of-The-World
npm install
```

### Usage

To run the project, execute the following command:

```sh
npm start
```

### Run tests

To run tests, run the following command:



```sh
  npm test
```

### Deployment

You can deploy this project using:

1. [Render](https://render.com/)
2. [AWS](https://aws.amazon.com/getting-started/hands-on/build-react-app-amplify-graphql/)
3. [Cpannel](https://dev.to/crishanks/deploy-host-your-react-app-with-cpanel-in-under-5-minutes-4mf6)
